📦 Database Setup (If Login Fails)

If you see "Invalid credentials" or "Login failed", the database needs to be set up:

Method 1: MySQL Workbench (Recommended - Easiest)

Step 1: Open MySQL Workbench
Step 2: Connect to your local MySQL server
Step 3: Click "Create new schema" button (database icon in toolbar)
Step 4: Name it: ali_hr_leave and click Apply
Step 5: Go to File → Run SQL Script
Step 6: Select: database\schema.sql and click Run
Step 7: Go to File → Run SQL Script (again)
Step 8: Select: database\seeds.sql and click Run
Done! Now refresh the login page and use the credentials above.

Method 2: Command Line

Open Command Prompt and run: 
mysql -u root -p
CREATE DATABASE ali_hr_leave;
USE ali_hr_leave;
SOURCE database/schema.sql;
SOURCE database/seeds.sql;
EXIT;

🔧 Troubleshooting

Problem: "Can't connect to MySQL server"
Solution: Start MySQL service:
  • Press Win+R, type services.msc
  • Find "MySQL80" or "MySQL"
  • Right-click → Start
Problem: Server not running
Solution: Double-click RUN_ME_NOW.bat in the project folder
Problem: Port 3000 already in use
Solution: Close other applications using port 3000, or run on different port: npm run dev -- -p 3001
